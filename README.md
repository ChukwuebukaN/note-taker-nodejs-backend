<h1>Getting started</h1>

- [x] Recommended: `Node.js 14+` and `npm 6+`
- [x] Install dependencies: `npm install` or `yarn`
- [x] Start the server: `npm run start` or `yarn start`
- [x] To deploy, you need to make a build: `npm run build`
